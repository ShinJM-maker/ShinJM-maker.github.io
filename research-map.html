Detailed Research Map

Read the roadmap quickly first, then open the advanced diagram if you want full node-level structure.

Stage 1. Core Linguistic and Syntax 2021-2022

Focus: Language structure and machine reading foundations.

  • Syntactic analysis: dependency parsing, neural-symbolic modeling.
  • Deep semantics: table MRC, WordNet, continual learning.
↓
Stage 2. Applied LLM and Domain RAG 2023-2024

Focus: Industrial optimization for real-world systems.

  • LLM evaluation and architecture design in production settings.
  • Domain RAG pipelines for high-stakes industrial use cases.
↓
Stage 3. Multimodal Document Intelligence 2025-2026

Focus: Document structure recovery, hierarchical retrieval, and evidence assembly.

  • M3DocDep (CVPR 2026): multimodal dependency chunking.
  • HiKEY (ACL 2026): hierarchical retrieval framework.
  • MultiDocFusion (EMNLP 2025): hierarchical RAG pipeline.
↓
Stage 4. Structured Memory, World Models, and Interactive Agents 2027+

Focus: Agents that recover compact state abstractions from multimodal streams, maintain actionable memory, and plan coherently in open-ended environments.

  • Actionable ontologies for multimodal agent memory and planning.
  • Executable intermediate representations for natural-language agents.
  • Video-structured memory: chunking and graph representations for long-horizon reasoning.
  • Multimodal agent planning in open-ended environments.
